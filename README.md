# AI Tic-Tac-Toe

An AI-powered Tic-Tac-Toe game built using Python, Tkinter, and the Minimax Algorithm.

## Features

- Unbeatable AI using Minimax
- Interactive GUI built with Tkinter
- Game theory and recursive search
- Object-oriented design

## Tech Stack

- Python
- Tkinter
- Minimax Algorithm

## Project Structure

```
Tic-Tac-Toe/
├── main.py
├── game.py
├── ai.py
├── gui.py
├── assets/
└── .gitignore
```

## Run Locally

```bash
python main.py
```
